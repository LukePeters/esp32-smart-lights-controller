# LIFX ESP32 Macropad

My custom-built ESP32 macropad for controling my the LIFX lights in my home office.

I'm using [mpremote](https://github.com/micropython/micropython/tree/master/tools/mpremote) to upload firmware to the ESP32 board. I created a helper script to make it as easy as plugging the device into my computer and executing `./upload.sh`
